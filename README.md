IMGCraft v2 Windows Edition

Works just like the python version just you can use it without installing python. 

This tool extracts files from cd images (.img) into a folder which you can rebuild your new img from that folder after whatever modifications you have made.

Extracts & Builds Fast

Stability is Important!!!!

Building CD Images this way is alot better than opening an img and adding files and rebuilding. Building from the ground up minimizes the chance of corrupting the img.
